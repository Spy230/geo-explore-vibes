# Руководство по правовым документам для сайта

## Обзор

Этот проект содержит полный набор правовых документов, необходимых для соблюдения российского законодательства при ведении онлайн-бизнеса.

## Созданные документы

### 1. Публичная оферта (`/legal/terms`)
- **Назначение**: Официальное предложение заключить договор
- **Размещение**: Чекбокс в формах заказа + ссылка в футере
- **Обязательность**: Да, для всех коммерческих сайтов

### 2. Политика конфиденциальности (`/legal/privacy`)
- **Назначение**: Информирование о сборе и обработке персональных данных
- **Размещение**: Ссылка в футере + упоминание в чекбоксе согласия
- **Обязательность**: Да, при сборе любых персональных данных

### 3. Согласие на обработку ПДн (`/legal/data-processing`)
- **Назначение**: Получение согласия на обработку персональных данных
- **Размещение**: Отдельный чекбокс в формах + ссылка в футере
- **Обязательность**: Да, обязательный чекбокс

### 4. Согласие на рекламную рассылку (`/legal/marketing`)
- **Назначение**: Получение согласия на отправку рекламных сообщений
- **Размещение**: Отдельный опциональный чекбокс + ссылка в футере
- **Обязательность**: Нет, но необходимо для рассылок

## Компоненты

### ConsentCheckboxes
Готовый компонент с правильно оформленными чекбоксами согласий:

```tsx
import ConsentCheckboxes from "@/components/legal/ConsentCheckboxes";

<ConsentCheckboxes
  onConsentChange={(consents) => setConsents(consents)}
  showMarketing={true} // показывать ли чекбокс рекламной рассылки
/>
```

### Footer
Футер с ссылками на все правовые документы:

```tsx
import Footer from "@/components/layout/Footer";

<Footer />
```

### CookieBanner
Баннер для получения согласия на использование cookies:

```tsx
import CookieBanner from "@/components/legal/CookieBanner";

<CookieBanner />
```

### OrderForm
Пример формы заказа с правильными согласиями:

```tsx
import OrderForm from "@/components/forms/OrderForm";

<OrderForm />
```

## Правила размещения

### ✅ Правильно:

1. **Разделенные чекбоксы**:
   - Отдельный чекбокс для оферты
   - Отдельный чекбокс для согласия на обработку ПДн
   - Отдельный чекбокс для рекламной рассылки (опционально)

2. **Кликабельные ссылки**:
   - Все документы доступны по прямым ссылкам
   - Ссылки открываются в новой вкладке

3. **Размещение в футере**:
   - Все документы доступны на каждой странице
   - Четкая структура и навигация

4. **Получение согласий ДО сбора данных**:
   - Согласия получаются перед отправкой формы
   - Обязательные согласия блокируют отправку без них

### ❌ Неправильно:

1. **Объединение согласий в один чекбокс**
2. **Заранее проставленные галочки**
3. **Скрытые или недоступные ссылки на документы**
4. **Сбор данных без получения согласий**
5. **Отсутствие cookie-баннера**

## Настройка для вашего проекта

### 1. Обновите контактную информацию

В каждом документе замените плейсхолдеры:
- `[название компании/ИП]` → ваше название
- `[адрес]` → ваш адрес
- `[телефон]` → ваш телефон
- `[email]` → ваш email
- `[ИНН]` → ваш ИНН
- `[ОГРН]` → ваш ОГРН

### 2. Адаптируйте под ваши услуги

В публичной оферте:
- Обновите описание услуг
- Укажите актуальные цены и условия
- Добавьте специфичные для вашего бизнеса условия

### 3. Настройте политику конфиденциальности

- Укажите, какие именно данные вы собираете
- Опишите цели обработки данных
- Добавьте информацию о третьих лицах, если передаете данные

### 4. Интегрируйте компоненты

```tsx
// В вашем основном компоненте
import Footer from "@/components/layout/Footer";
import CookieBanner from "@/components/legal/CookieBanner";

function App() {
  return (
    <div>
      {/* Ваш контент */}
      <Footer />
      <CookieBanner />
    </div>
  );
}
```

```tsx
// В формах заказа
import ConsentCheckboxes from "@/components/legal/ConsentCheckboxes";

function OrderForm() {
  const [consents, setConsents] = useState({
    terms: false,
    privacy: false,
    marketing: false,
  });

  const handleSubmit = (e) => {
    if (!consents.terms || !consents.privacy) {
      alert("Необходимо дать согласие на обработку данных");
      return;
    }
    // Отправка формы
  };

  return (
    <form onSubmit={handleSubmit}>
      {/* Поля формы */}
      
      <ConsentCheckboxes
        onConsentChange={setConsents}
        showMarketing={true}
      />
      
      <button type="submit">Отправить</button>
    </form>
  );
}
```

## Важные замечания

1. **Статус оператора ПДн**: Не забудьте подать уведомление в Роскомнадзор
2. **Обновление документов**: Регулярно проверяйте актуальность документов
3. **Хранение согласий**: Ведите учет полученных согласий с датами
4. **Отзыв согласий**: Предусмотрите механизм отзыва согласий

## Штрафы за нарушения

- **Физические лица**: до 75 000 рублей
- **Юридические лица**: до 300 000 рублей
- **Реклама без согласия**: до 500 000 рублей для юр. лиц

## Поддержка

При возникновении вопросов обратитесь к юристам, специализирующимся на защите персональных данных и интернет-праве.
